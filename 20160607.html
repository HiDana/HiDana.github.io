解決MAMP PRO Forbidden問題

php mamp mac code

不知道為什麼安裝pro版畫面打localhost:8888後會顯示這樣

Forbidden

You don't have permission to access / on this server.

免費版卻可以直接以htdocs的資料夾型態來呈現

就覺得很討厭，但是在後面輸入當案名稱（localhost:8888/檔案名稱）就可以運行

一種好拉好拉的無奈感覺也持續好一段時間

直到最近，才有時間去把這問題解決

結果沒想到超簡單的....

1.打開mama pro

2.點擊Hosts

3.點擊Extended

4.選取下方Indexes

5.按下save

耶！！！！這樣一來就可以離開打網址的日子了
