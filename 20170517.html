Deploy react on github

code react github

寫完react就想要部署在一個有網址的地方

因為此部署是建立在 create react app 上的，所以先確認電腦有 create-react-app 的指令

$ npm install -g create-react-app 
$ create-react-app <project name>
$ cd <project name>

在package.json檔裡加入

"homepage": "https://<github-username>.github.io/<project-repo>"

比如說我的GitHub username - hidana 放在一個叫做 create-react-app-GitHubDeploy 的 repository 裡，就會長這樣

"homepage": "https://hidana.github.io/create-react-app-GitHubDeploy"

#package.json
{
  "name": "deploygithub",
  "version": "0.1.0",
  "private": true,
  "homepage": "https://hidana.github.io/create-react-app-GitHubDeploy", #add this
  "dependencies": {
    "react": "^15.5.4",
    "react-dom": "^15.5.4"
  },
  "devDependencies": {
    "react-scripts": "0.9.5"
  },
  "scripts": {
    "start": "react-scripts start",
    "build": "react-scripts build",
    "test": "react-scripts test --env=jsdom",
    "eject": "react-scripts eject"
  }
}

如果此時下

$ npm run build

就會出現建議你要安裝 gh-pages

$ npm install --save-dev gh-pages

並在 package.json 的 scripts 加入

"scripts": {
    // ...
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build"
  }

切記！！！

在 deploy 之前 一定要先 commit！！！因為deploy會自動部署在github上

$ git init
$ git add .
$ git commit -m "deploy create react app on github"
$ git remote add origin https://github.com/HiDana/create-react-app-GitHubDeploy.git

然後再下

$ npm run deploy

可以看到github上面已經部署build的檔案了

到setting上看到他已經自動部署在 github 上了

#final package.json
{
  "name": "deploygithub",
