有介於設計與前端之間的 specification 溝通

因此開發一個可供應設計與前端的簡單spec

設計可於左側輸入color與font，右側body即會產生spec

開發中遇到幾個蠻有趣的技術與大家分享

  • color及font的斷點是段在每一句的分號，再用push去合併成一個陣列
colorText.split(';').forEach(
      function(obj,index) {
        result.push({
          name: obj.split(':')[0],
          color: obj.split(':')[1]
        })
      }
    )
  • : before 的 setStyle

在於左側輸入框的顏色會依照使用者輸入的顏色而做變動

因為輸入框上面的方框是用 : before 壓在上面的

網路上大部分都是運用 jquery 去把它寫出來

所以在 純js 設定時，有別於平常的寫法去把before抓出來

document.getElementById("sidebar_color")
        .appendChild(document.createElement("style"))
        .innerHTML = `#sidebar_color:before {background: ${resultArray[0].color};}`
